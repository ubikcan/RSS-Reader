This is a side-project of my AAAS Fellowship at DOE. The goal is to create a web-based app that automatically gathers subscribed RSS newsfeeds from DOE National Labs, specifically OLCF, ALCF, and NERSC.

Staging:

&#9989; Where RSS feeds are provided, allow a logged-in user to add the RSS feeds (or theoretically any other RSS feed).

&#9989; Allow the user to unsubscribe from feeds.

&#9989; Where RSS feeds do not exist, either add a xml feed (eg created in rss.app) or make a bespoke scrape capability (if allowed).
